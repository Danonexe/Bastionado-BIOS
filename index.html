La BIOS (Basic Input/Output System) o su versión moderna UEFI (Unified Extensible Firmware Interface) es el primer software base que se ejecuta al encender el ordenador.
Su función principal es reconocer el hardware instalado y gestionar el arranque del sistema operativo.

Si no configuramos bien este sofware, un atacante con acceso físico podría modificar el orden de arranque, usar un USB con herramientas externas o incluso restablecer la BIOS para obtener acceso al equipo.

En esta guía se muestran los pasos para bastionar la BIOS/UEFI, es decir, configurarla para maximizar la seguridad del dispositivo.

2.1. Establecer contraseña de administrador

  1. Enciende el equipo y accede a la BIOS/UEFI pulsando la tecla correspondiente en mi caso es el F10.
  2. En el menú principal, buscamos la pestaña Security.
  3. Selecciona Set Administrator Password o Supervisor Password.
  4. Introduce una contraseña robusta (mínimo 12 caracteres, combinando letras, números y símbolos).
  5. Siempre que hagamos un cambio en la BIOS tendremos que guardar los cambios.

Esto hará que siempre te pida la contraseña antes de modificar cualquier parámetro de la BIOS.

2.2. Establecer contraseña de usuario o de arranque (Power-On Password)

  1. Desde el mismo menú Security, selecciona Set User Password o Power-On Password.
  2. Escribe una contraseña distinta a la de administrador.

Esta contraseña te la pedirá en cada inicio, impidiendo que terceros arranquen el equipo sin autorización. En la imagen anterior se puede ver donde cambiar la User Password.

2.3. Deshabilitar el arranque desde dispositivos externos

Como dehabilitar el arranque desde dispositivos externos:

  1. Accede al menú Boot o Arranque.
  2. Busca la opción Boot from USB, External Devices o similar.
  3. Cambia su valor a Disabled.

Con estos cambios el equipo no podrá iniciarse desde memorias USB o DVD con sistemas no autorizados.

2.4. Configurar un orden de arranque seguro

  1. Dentro del menú Boot Priority o Boot Order, verifica los dispositivos listados.
  2. Coloca el disco duro o SSD principal como primera opción.
  3. Desactiva (o coloca al final) otros medios de arranque como CD/DVD o USB.

Le da la prioridad al ordenador de solo arrancar desde el disco donde está instalado el sistema operativo.

2.5. Activar Secure Boot

  1. En el menú Boot → Secure Boot, cambia el estado a Enabled.
  2. Si aparece la opción Key Management, selecciona Install Default Keys para usar las claves oficiales del fabricante.

Esto solo te permitirá arrancar software firmado y verificado digitalmente, como Windows o distribuciones Linux compatibles.

2.6. Otras opciones de seguridad recomendadas

  • TPM (Trusted Platform Module): Activa esta función si está disponible. Permite almacenar claves de cifrado y mejora la seguridad del arranque.
  • Desactivar Wake-on-LAN: Evita que el equipo se encienda de forma remota.
  • Bloqueo físico de la carcasa: Algunos equipos permiten proteger con llave el acceso interno para evitar manipulaciones (como quitar la pila CMOS).
  • Guardar los cambios solo con contraseña: En algunas BIOS puedes exigir la contraseña del administrador antes de guardar cualquier cambio.

Aunque la configuración del firmware aumenta la seguridad, sigue siendo importante proteger el equipo físicamente y guardarlos en sitios seguros.

Una cosa a tener en cuenta es que algunas placas base permiten borrar la configuración de la BIOS retirando la pila CMOS o mediante pines Clear CMOS.

Con estos pasos se logra un bastionado eficaz de la BIOS/UEFI, reduciendo drásticamente las posibilidades de que un atacante acceda al sistema o manipule el proceso de arranque.
Aunque ninguna medida es infalible, una correcta configuración del combinada con cifrado de disco y medidas físicas constituye una defensa sólida frente a accesos no autorizados.
